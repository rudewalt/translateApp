## TranslateApp
Тестовое задание в школу мобильной разработки Яндекс (https://academy.yandex.ru/events/mobdev/msk-2017/)

### Используемые библиотеки
* _Dagger 2_
* _RxJava 2_
* _OkHttp 3_
* _Retrofit 2_
* _ButterKnife_

#### Тестирование
* _Assertj_
* _Mockito_
